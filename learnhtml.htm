1. Introduction to HTML

HTML stands for Hypertext Markup Language. It is the standard language for creating webpages.

1.1 What is HTML?

HTML stands for "HyperText Markup Language" and is used to create web pages. It is the foundation of any website on the internet.

1.2 The <head> Section

The <head> section contains information that is not directly visible to users, such as metadata, titles, and stylesheet links. Example:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>

This sets the character encoding, viewport settings, and page title.

1.3 The <body> Section

The <body> section contains all the visible content you see on the page, such as text, images, and headings. Example:

<body>
    <h1>Welcome to My Page</h1>
    <p>This is a text paragraph on the page.</p>
</body>

This is where the content displayed to users is placed.

1.4 The <title> Element

The <title> element specifies the title of the page that appears in the browser's title bar or tab. Example:

<title>Page Title</title>

This title helps users understand the content of the page and provides important information for search engines.

2. Basic HTML Elements

2.1 Heading

Headings are defined with the <h1> to <h6> tags:

<h1>This is a heading 1</h1>
<h2>This is a heading 2</h2>
<h3>This is a heading 3</h3>
<h4>This is a heading 4</h4>
<h5>This is a heading 5</h5>
<h6>This is a heading 6</h6>

2.2 Paragraph

Paragraphs are defined with the <p> tag:

<p>This is a paragraph.</p>

2.3 Link

Links are defined with the <a> tag:

<a href="https://www.example.com">This is a link</a>

2.4 Image

Images are defined with the <img> tag:

<img src="image.jpg" alt="Description of image">

2.5 List

HTML supports ordered and unordered lists:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

2.6 Table

Tables are defined with the <table>, <tr>, <th>, and <td> tags:

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>

2.7 Form

Forms are defined with the <form> tag and include various input types:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Submit">
</form>

2.8 Div and Span

Use <div> for block-level grouping and <span> for inline grouping:

<div>This is a block-level container.</div>
<span>This is an inline container.</span>

2.9 Semantic HTML

Semantic HTML elements provide meaning to the structure of a web page:

<header>Header content</header>
<nav>Navigation links</nav>
<main>Main content</main>
<footer>Footer content</footer>

2.10 Media

Adding videos and audio to your page:

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

3. HTML Attributes

3.1 Class and ID

Classes and IDs are used for styling:

<div class="my-class">Content</div>
<div id="my-id">Content</div>

3.2 Style Attribute

The <style> attribute is used for inline CSS:

<p style="color: red;">This is a red paragraph.</p>

3.3 Target Attribute

The <target> attribute specifies where to open the linked document:

<a href="https://www.example.com" target="_blank">Open in a new tab</a>

3.4 Placeholder Attribute

The <placeholder> attribute provides a hint within an input field:

<input type="text" placeholder="Enter your name">

3.5 Required Attribute

The <required> attribute forces users to fill out the field:

<input type="text" required>

3.6 Multiple Attribute

The <multiple> attribute allows multiple file selections:

<input type="file" multiple>

3.7 Action Attribute

The <action> attribute specifies where to send form data:

<form action="submit.php"></form>

3.8 Method Attribute

The <method> attribute specifies the HTTP method to use:

<form method="post"></form>

3.9 Enctype Attribute

The <enctype> attribute specifies how form data is encoded:

<form enctype="multipart/form-data"></form>

3.10 Value Attribute

The <value> attribute specifies the value of an input:

<input type="text" value="Default text">

4. HTML Forms

4.1 Text Input

The <input> tag with type="text" creates a single-line text input:

<input type="text">

4.2 Password Input

The <input> tag with type="password" creates a password field:

<input type="password">

4.3 Checkbox Input

The <input> tag with type="checkbox" creates a checkbox:

<input type="checkbox"> Check me

4.4 Radio Input

The <input> tag with type="radio" creates a radio button:

<input type="radio" name="group" value="option1"> Option 1<br>
<input type="radio" name="group" value="option2"> Option 2

4.5 Textarea

The <textarea> tag defines a multi-line text input:

<textarea rows="4" cols="50">Default text</textarea>

4.6 Select Box

The <select> tag creates a dropdown list:

<select>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
</select>

4.7 Button

The <button> tag creates a clickable button:

<button type="button">Click me</button>

4.8 Fieldset

The <fieldset> tag groups related elements:

<fieldset>
  <legend>Personal Information</legend>
  <label for="name">Name:</label>
  <input type="text" id="name">
</fieldset>

4.9 Legend

The <legend> tag represents a caption for a <fieldset>:

<fieldset>
  <legend>Details</legend>
  <input type="text">
</fieldset>

5. Media Elements

5.1 Video

The <video> tag embeds a video:

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

5.2 Audio

The <audio> tag embeds an audio file:

<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  Your browser does not support the audio element.
</audio>

5.3 Iframe

The <iframe> tag defines an inline frame:

<iframe src="https://www.example.com" width="600" height="400"></iframe>

6. Styling HTML

6.1 Inline CSS

Inline styles use the <style> attribute:

<p style="color: blue; font-size: 20px;">Styled text</p>

6.2 Internal CSS

Internal styles are defined within <style> tags in the head:

<style>
  p { color: green; font-size: 18px; }
</style>

6.3 External CSS

External styles are defined in a separate .css file:

<link rel="stylesheet" href="styles.css">

6.4 Color

Change text color using CSS:

p { color: red; }

6.5 Font Size

Change font size using CSS:

p { font-size: 20px; }

6.6 Font Weight

Make text bold using CSS:

p { font-weight: bold; }

6.7 Font Style

Make text italic using CSS:

p { font-style: italic; }

6.8 Text Decoration

Apply underline, overline, or line-through:

p { text-decoration: underline; }

6.9 Text Align

Align text to left, right, or center:

p { text-align: center; }

6.10 Background Color

Set background color for elements:

div { background-color: lightgrey; }

6.11 Margin

Add space outside an element:

div { margin: 20px; }

6.12 Padding

Add space inside an element:

div { padding: 20px; }

6.13 Border

Add border to an element:

div { border: 2px solid black; }

6.14 Border Radius

Round the corners of an element:

div { border-radius: 10px; }

6.15 Display

Control the display of elements:

div { display: none; }

6.16 Flexbox

Use Flexbox to create flexible layouts:

div { display: flex; justify-content: center; }

6.17 Grid

Create a grid layout using CSS Grid:

div { display: grid; grid-template-columns: 1fr 1fr; }

6.18 Box Shadow

Add shadow to elements:

div { box-shadow: 5px 5px 10px grey; }

6.19 Text Shadow

Add shadow to text:

p { text-shadow: 2px 2px 5px grey; }

6.20 Font Family

Change the font of text:

p { font-family: Arial, sans-serif; }

6.21 Line Height

Adjust the line height of text:

p { line-height: 1.5; }

6.22 Text Transform

Control text capitalization:

p { text-transform: uppercase; }

6.23 White Space

Control whitespace handling:

p { white-space: nowrap; }

6.24 Overflow

Control overflow content handling:

div { overflow: auto; }

6.25 Visibility

Control element visibility:

div { visibility: hidden; }

7. Advanced HTML & CSS

7.1 Media Queries

Apply styles based on viewport size:

@media (max-width: 600px) {
  body { background-color: lightblue; }
}

7.2 CSS Variables

Define and use CSS variables:

:root {
  --main-bg-color: coral;
}
body { background-color: var(--main-bg-color); }

7.3 Custom Fonts

Include custom fonts using @font-face:

@font-face {
  font-family: 'CustomFont';
  src: url('custom-font.woff2') format('woff2');
}
body { font-family: 'CustomFont', sans-serif; }

7.4 Flexbox Alignment

Align items using Flexbox:

div { display: flex; align-items: center; justify-content: space-between; }

7.5 Grid Layout

Define grid areas and positions:

.container {
  display: grid;
  grid-template-areas: "header header"
                       "sidebar content"
                       "footer footer";
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }
